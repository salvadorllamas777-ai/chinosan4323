# proyecto2 — Node + MongoDB scaffold

Minimal scaffold with Express and Mongoose.

Quick start:

```bash
npm install
cp .env.example .env
# adjust .env if needed
npm run dev
```

Endpoints:
- `GET /health` — simple health check

Optional with Docker:

```bash
docker-compose up --build
```
